# Style Examples

This folder contains example styles to use as references when building new components.

## Files

- `button-examples.module.css` - Multiple button variants (solid, outline, ghost, text)
- `card-examples.module.css` - Card layouts (basic, with image, interactive)
- `typography-examples.module.css` - Heading scales, text utilities
- `layout-examples.module.css` - Grid, flex patterns, containers

## Usage

Copy patterns from these examples when creating new components. Modify the design tokens in `../styles/variables.css` to change colors, spacing, or typography globally.
